# Security policy

No publiques issues con credenciales, rutas privadas, prompts sensibles o resultados que contengan datos personales.

El benchmark funcional no debe modificarse para ejecutar shell real sin una revisión de seguridad específica. Las pruebas con herramientas reales deben usar un repositorio o entorno separado, mínimo privilegio y confirmación humana fuera del contexto del modelo.
